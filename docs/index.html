SimpleRecommender

Get instant movie recommendations using Cosine Similarity.

View on GitHub Download Code

How It Works

  • Load the Netflix watch‑history CSV.
  • Build a user‑movie matrix (implicit ratings).
  • Compute cosine similarity between movies.
  • Return the top‑N most similar titles.
